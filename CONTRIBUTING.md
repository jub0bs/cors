# Contributing to jub0bs/cors

jub0bs/cors is an open-source project but currently does not accept external
contributions. My reasons echo [Dominik Honnef's][dominikh].

To report a problem that doesn't affect security
(such as a harmless bug, a missing feature, a misfeature, an idea for
performance improvement, etc.), please [open an issue][new-issue];
To report a problem that does affect security,
please [file a security advisory][new-advisory].

[dominikh]: https://github.com/dominikh/go-tools?tab=contributing-ov-file#pull-requests
[new-advisory]: https://github.com/jub0bs/cors/security/advisories/new
[new-issue]: https://github.com/jub0bs/cors/issues/new
